# Compiladores - Prova 1 


## Calculadora com comandos de atribuição, print e operações sobre strings de caracteres.

A prova é uma extensão do exercício 2 (E2), agora
com leitura de arquivo de entrada com comandos e expressões.

Todos os testes do E2 devem passar, mas não contabilizam para a nota final.
Apenas a implementação das extensões abaixo serão consideradas para fins de avaliação na prova 1.


## Extensões


- comentários de uma linha começando por \\
- strings de caracteres (caracteres entre aspas duplas) de uma linha
- uso de variáveis
- comando de atribuição
- comando print para imprimir strings e valores inteiros
- operações pré-definidas sobre strings:
   - concat(s1, s2, ...)  – concatena duas ou mais strings.
   - length(s)  – retorna tamanho da string s.


## Passos


Usar arquivos de E2, renomeando para prova1.l e prova2.y.

A pasta de trabalho é ```P1```. Usar apenas essa pasta.

- Utilizar o arquivo P1/main.c (não modificar).

- Utilizar o makefile ou comandos que estão no makefile.

- Rodar:

./prova1 exemplo.in exemplo.out
